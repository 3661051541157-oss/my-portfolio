ผมชื่อนายนันทิพัฒน์ รุ่งโรจน์

นักศึกษาชั้นปีที่ 3 สาขาเทคโนโลยีทางธุรกิจดิจิทัล มีความสนใจด้านเทคโนโลยีและการพัฒนาเว็บไซต์ สามารถพัฒนางานด้วยภาษา HTML, CSS และ PHP รวมถึงมีความรู้พื้นฐานด้านการเขียนโปรแกรม นอกจากนี้ยังสามารถใช้โปรแกรม Microsoft Office ได้แก่ Word, Excel และ PowerPoint รวมถึง Canva ในการจัดทำเอกสาร งานนำเสนอ และการออกแบบเบื้องต้นได้ พร้อมเรียนรู้สิ่งใหม่ ๆ และพัฒนาทักษะของตนเองอย่างต่อเนื่องเพื่อเตรียมความพร้อมสำหรับการทำงานในอนาคต
